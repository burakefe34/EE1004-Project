# EE1004 Group 6 – Library Management System

This project was developed for the EE1004 Java Programming course.

The application allows a small library to manage books, magazines, and DVDs. Users can add items, list available items, check out items, return items, and search by title.

## Requirements

- JDK 17 or newer

## Compile

Open a terminal in the `LibraryManagement_Code` folder and run:

```bash
javac *.java
```

## Run (Console Version)

```bash
java Main
```

## Run (GUI Version)

```bash
java LibraryGUI
```

## Project Files

- LibraryItem.java
- Book.java
- Magazine.java
- DVD.java
- Library.java
- Main.java
- LibraryGUI.java

## Features

- Add new library items
- List all items
- Check out items
- Return items
- Search items by title
- Different loan periods for Books, Magazines, and DVDs
- Graphical User Interface (GUI)

## OOP Concepts Used

- Abstraction
- Inheritance
- Polymorphism
- Encapsulation
- Aggregation
